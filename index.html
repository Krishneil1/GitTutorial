Basic Github Commands

    create a working copy of a local repository by running the command.

  • git clone /path/to/repository

    • when using a remote server, your command will be.

  • git clone username@host:/path/to/repository e.g git clone krishneil1@host:/path/to/repository

    • Next is find the Status.

  • git status. This will give status of any new files that has not been committed

    • Lock the files to be committed.

  • git commit add index.html . This sync the files but not yet added to your remote git repository

    • Add files to remote repository.

  • git push . This will up load the files to GitHub

    • Add All file/files to remote repository.

  • git add * or git add -A Adds all the files in Directory
